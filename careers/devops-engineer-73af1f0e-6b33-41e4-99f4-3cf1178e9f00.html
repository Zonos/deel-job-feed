About the Role

Zonos is seeking a DevOps Engineer with 3+ years of experience to join our team. This position will be dedicated to supporting Zonos' cloud infrastructure, automation, and new Engineering initiatives.

What You'll Work On

  • Contribute to and maintain infrastructure using Terraform;
  • Create and manage automation workflows using Github Actions and Jenkins;
  • Monitor application uptime/performance and generate actionable recommendations;
  • Help maintain data security policies and best practices;
  • Implement testing at different stages of the development cycle;

What We're Looking For

  • AWS certified and/or extensive experience with AWS services;
  • Linux system administration;
  • MySQL and PostgreSQL database administration;
  • Webserver administration;
  • Writing and managing AWS IAM policies;
  • Setting up secure network access with AWS Security Groups;
  • Configuring CI/CD workflows;
  • Managing codebases with git and Github actions;
  • Instrumenting observability tools for API health/performance;
  • Docker and Kubernetes/ECS;
